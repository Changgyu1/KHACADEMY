package란
- 어떠한 폴더 위치를 작성하는 공간
- 내가 만들고싶은 파일 경로(폴더 위치)를 설정할 수 있음


카멜 케이스 - 자바 명명 규칙
- 자바의 명명 규칙은 클래스나 패키지 변수 상수 메소드 등
- 자바 프로그래밍을 할 때 이름을 따라할 규칙

카멜 케이스 (ex: javaName, myName, firstName)
- 이름에 여러 단어를 혼합해서 사용한다면 첫 번째 단어는 소문자, 두 번째 단어부터 대문자로 사용
- 이름을 지울 때 숫자를 먼저 입력해도 안됨 (ex: 1004nany, 1004Nany) - 무조건 영어먼저

스네이크 케이스 (ex: java_name, my_name, first_name)
- 모든 문자의 첫 자는 소문자로 표기하고 문자들 사이에는 _(언더바)로 연결

파스칼 케이스 (ex: JavaName, MyName, FirstName)
- 모든 문자의 첫 자를 대문자로 표기 , 중간에 대문자를 넣어 표기

케밥 케이스 (ex:java-name, my-name, first-name)
- 모든 문자의 첫 자는 소문자로 표기하고 문자들 사이에는 -(하이픈) 연결
